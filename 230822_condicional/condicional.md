Una de las ideas claves en programación es la de condicionales.

Se trata de hacer que el programa ejecute unas instrucciones u otras dependiendo de alguna situación o condición.

if ( 2 < 5 ){ // lo que esta escrito dentro de los paréntesis es cierto

// por tanto, aquello que esté escrito acá se ejecuta

}

if ( 2 == 5 ){ // esto es falso (el doble == sí significa 'igual que')...

// por tanto, aquello que esté escrito acá NO se ejecuta

}

Este ejemplo hace que se prenda un LED rojo o uno Azul dependiendo de si el pulsador está oprimido o no.
